# Social Media Backend

This is the backend for a social media application built with Node.js, Express, and MongoDB.

## Features
* **User Accounts:** Signup, Login, and Logout.
* **Profiles:** Update profile info, and Follow/Unfollow other users.
* **Posts:** Create new posts with images, Like, Comment, and Delete.
* **Notifications:** Get alerts for likes, comments, and follows.
* **AI Chatbot:** Interactive assistant powered by Google Gemini AI.

## How to Run

1. **Install packages:** Open your terminal in this folder and run:
   ```bash
   npm install
   ```

2. **Set up environment variables:** Create a new file named `.env` in this folder and add your details:

3. **Start the server:** Run this command to start the backend:
   ```bash
   npm run dev
   ```
